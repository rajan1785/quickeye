# QuickEye
Few-shot visual QA for MSME factories. Turn any Android phone into a custom defect detector in 5 minutes.

## What it does
Worker captures 20 OK + 20 defective units. The phone becomes the QA station — no labeling, no fine-tuning, no IT team.

## Stack
- Frontend: Next.js 14 PWA + Capacitor (Android APK)
- Backend: FastAPI on Modal Labs
- Vision: DINOv3 embeddings + GPT-5.4-mini vision agent
- QC guides: GPT-5.5 (English + Hindi)
- Storage: Supabase
